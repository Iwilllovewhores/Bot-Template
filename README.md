# 🤖 Bot-functions

Um bot para Discord desenvolvido com Discord.js v14, focado em fornecer funções utilitárias, organização modular e fácil expansão.

---

## 🚀 Funcionalidades

- Sistema de comandos modular
- Sistema de eventos separado
- Suporte a slash commands
- Estrutura pronta para expansão

---
## ⚙️ Instalação

git clone https://github.com/Iwilllovewhores/Bot-Template.git
cd Bot-functions
npm install

---

## 🔧 Configuração

Token=

---

## ▶️ Executar

node .

ou

npm run dev

---

## 🧠 Como funciona

- Commands → comandos do bot
- Events → eventos automáticos
- Functions → funções reutilizáveis

---

## 📌 Tecnologias

- Node.js
- Discord.js v14
- JavaScript

---

## 🛠️ Objetivo

Base simples e modular para criação de bots Discord escaláveis.
